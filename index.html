Configuración inicial

Necesitas conectar Firebase para que funcione en la nube. Es gratis y tarda ~5 minutos.

1
Ve a console.firebase.google.com → crea un proyecto.
2
Realtime Database → Crear → europe-west1 → Modo prueba.
3
Authentication → Método de acceso → Correo/Contraseña → Activar. (Necesario para el login de clientes.)
4
⚙ Config del proyecto → General → </> Web → copia firebaseConfig y pégalo en este archivo HTML.
